# Flow: Define New Feature

**Actor:** Product Manager

<execution_protocol>

1. **Ingest:** Ask the user for the high-level goal.
2. **Draft:** Create a new file in `.windsurf/memory/pm/[feature_name].md`.
3. **Structure:** Fill the file with:
   - **Vision:** The "Why".
   - **User Stories:** `As a... I want... So that...`
   - **Acceptance Criteria:** Gherkin syntax (`Given/When/Then`).
4. **Verify:** Ask user to confirm the scope before saving.
   </execution_protocol>
