# Flow: Define Data Schema

**Actor:** Business Analyst / QA

<execution_protocol>

1. **Input:** Read the active Feature Spec in `.windsurf/memory/pm/`.
2. **Model:** Define the data structure (JSON/SQL) required to support the feature.
3. **Validate:** Check for normalization and data types.
4. **Output:** Write schema definition to `.windsurf/memory/analyst/[feature]_schema.md`.
   </execution_protocol>
